# 🌍 AI Multi-Language Translator

A simple Python desktop application that translates text between 24 languages using Google Translate via Deep Translator.

## Features

* Translate text instantly
* Auto language detection
* 24 supported languages
* Copy translated text
* Swap languages
* Translation history with timestamps
## Technologies Used

* Python
* Tkinter
* Deep Translator
  
 How It Works
1. Enter text in the input box.
2. Select source language (or Auto Detect).
3. Select target language.
4. Click Translate.
5. View translated text instantly.
Translation history is automatically saved.
